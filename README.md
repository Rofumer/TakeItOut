## ⚡ TakeItOut Minecraft Mod — Fast Shulker Block Picking + Easy Place

Pick blocks directly from **shulker boxes — whether they’re in your inventory or placed in the world** — using:

- the **pick block key (mouse wheel)**  
- **Litematica Easy Place (right-click in schematics)**  
- or **regular right-click in schematics with Auto Take Out enabled**

Build faster than ever — whether you use Easy Place or place blocks manually.

Works seamlessly with **Litematica schematics** and is especially useful with any Litematica printer.

## ✨ Key Features

- 📦 Pick blocks straight from **shulker boxes in your inventory or placed in the world**

- 🧱 **Litematica Easy Place support**  
  Right-click schematic blocks to auto-pick and place them exactly as defined by the schematic

- 🖱️ **Manual placement support (Auto Take Out)**  
  Right-click in schematics to automatically take the required item —  
  blocks are placed normally (as the player would place them)

- ⚡ Works with **Litematica schematics**  
- 🖱️ Pick block support — use the **mouse wheel** to grab blocks from shulkers  
- 🔄 Auto pick mode (toggle with **R**)  
- 🖨️ Perfect for fast building with **Litematica printers**  
- 🔄 Support for **Tweakeroo Restock**  
- 📦 **Extraction Modes** — SINGLE ITEM / FULL STACK (switch with **B**)

## 🆕 What's New

- 📦 **World Container Sources**  
  Link placed chests, barrels, and shulkers using the configurable **Link Looked-at Container** hotkey (**H** by default).  
  The mod can now pull items directly from the world when they are not in your inventory.

- 🌐 **Cross-Container Item Access**  
  Automatically uses linked containers if required items are missing from your inventory or carried shulkers.

- 🌍 **Cross-Dimension Support**  
  Linked containers can now be used across dimensions (depending on server settings).

- 📊 **Litematica Material List Integration**  
  Linked containers now contribute to available materials in the Litematica material list.

- 🔤 **Item Sorting**  
  Sort items by **Name** or **Count** in the item list.

- 🧭 **Container Management UI**  
  - **All Items** — view all available items  
  - **Containers** — manage and inspect linked containers  

- 🟢 **Container Outline Rendering**  
  Highlight linked containers in the world.

- ⚙️ **Full Settings Screen**  
  Open with **X + T** or via Controls / Mod Menu.

- 🔄 Improved container linking behavior  
  Containers persist per world/dimension and can be managed individually.

## ⚠️ Compatibility Notice

- ❗ **Conflict with Item Swapper**  
  The mod may conflict with Item Swapper, as both modify item picking behavior.  
  This can lead to incorrect item selection or unexpected hotbar swaps.

👉 If you experience issues, try disabling Item Swapper or adjusting its settings.

## 🌐 Multiplayer Support

Install the mod on the **server** to use it on public servers  
or use the companion plugin for:

- Paper  
- Purpur  
- Spigot  
- Bukkit  

### ⚠️ Compatibility Notice

Older client versions of TakeItOut are **not compatible** with the current server version.

This incompatibility was introduced after adding **Single Item Mode**.  
Using an outdated client may result in a **disconnect from the server**.

👉 Please update the mod to the latest version on the client.
